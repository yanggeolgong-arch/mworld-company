# AI KOREA DATA LAB (aikorea-datalab)

Data-Driven Culinary Intelligence.

[Next.js](https://nextjs.org) 기반 프로젝트.

## 시작하기

개발 서버 실행:

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속.

## 배포

- **Vercel**: `git push vercel main` — aikoreadatalab.com 기준 배포
- `.cursor/rules/mworld-executive-protocol.mdc` 프로토콜 준수
